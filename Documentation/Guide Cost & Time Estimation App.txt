# Cost & Time Estimation App

## Quick Setup and User Guide

Thank you for downloading and testing the Cost & Time Estimation App.

This solution was built using standard Microsoft Power Platform components to help estimate project cost, delivery time, resource availability, and generate professional quote documents.

The application is designed to work with SharePoint Lists and does not require premium connectors.

---

# Before You Start

## 1. Import the Solution

Import the Power Apps solution into your environment.

Ensure all SharePoint Lists are available and connected.

---

## 2. Import the Service Library

Import the provided Service Library CSV into the SharePoint list:

**Scope_Master_IT_Service_Library**

This contains the sample services used for estimation.

---

## 3. Import Sample Projects

Import the provided sample project data into:

**Scope_Projects**

This will allow you to test the full workflow immediately.

---

## 4. Add the New SharePoint Column

A new SharePoint column has been introduced.

### Column Name

```text
Discount(%)
```

### Type

```text
Number
```

### Decimal Places

```text
0
```

This column stores the project discount percentage and is automatically used throughout the quotation and pricing calculations.

Example:

| Discount(%) |
| ----------- |
| 0           |
| 5           |
| 10          |
| 15          |
| 20          |

The application will automatically calculate:

* Amount Saved
* Total After Discount
* GST
* Total Including GST

---

# Using the Solution

## Step 1

### Open the App

Launch the application.

Select:

**Start Estimating**

---

## Step 2

### Create or Select a Project

Choose an existing project or create a new one.

Enter:

* Project Name
* Client
* Start Date
* Weekly Available Hours
* Buffer Percentage
* Discount Percentage

---

## Step 3

### Build Your Estimate

Open the Services tab.

Select the required services from the Master Service Library.

The system will automatically:

* Add services
* Calculate hours
* Calculate cost
* Calculate project duration
* Update the estimate summary

---

## Step 4

### Review Timeline

Open the Timeline tab.

The system will estimate:

* Working hours
* Estimated days
* Estimated weeks
* Start date
* Completion date

A visual calendar is generated to help with delivery planning.

---

## Step 5

### Review Project Tracking

Select:

**Send to Project Tracker Board**

This creates a tracking record for the project.

The tracking board helps monitor:

* Active projects
* Delivery progress
* Project status
* Estimated completion

---

## Step 6

### Review Financials

Open the Financials tab.

The quotation automatically calculates:

* Base Estimate
* Buffer Allowance
* Standard Total
* Discount Amount
* Total After Discount
* GST
* Total Including GST

---

## Step 7

### Print or Email Quote

Use the Print button to generate a printable quotation.

Use the Email button to send the quotation to the client.

The generated quote includes:

* Client details
* Service breakdown
* Timeline
* Cost summary
* GST
* Final project value

---

# Typical Workflow

