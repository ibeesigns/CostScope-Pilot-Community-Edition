# CostScope Pilot Community Edition

A free community project for project estimation, pricing, timeline planning, and quote generation.

Built using standard Microsoft Power Platform components to demonstrate what can be achieved without premium connectors.

## Included

* Power Apps Solution
* SharePoint Templates
* Sample Project Data
* Installation Guide

## Why I Shared This

This project was originally built to solve a practical problem: understanding project cost, effort, and delivery timelines before work begins.

Many organisations already have access to powerful Microsoft tools but often invest in additional software without first exploring what is possible.

The goal of sharing this project is to encourage learning, experimentation, and practical solution building.

## Community Edition

This version is intentionally shared as a learning resource.

Some controls and components remain close to their original build structure to make it easier for others to explore the solution and understand how everything works together.

## Getting Started

1. Import the Power Apps solution.
2. Create the SharePoint Lists from the supplied templates.
3. Import the sample data.
4. Open the application and start estimating.
