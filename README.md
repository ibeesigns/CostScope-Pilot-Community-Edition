# CostScope Pilot Community Edition

A free Microsoft Power Platform solution for project estimation, pricing, timeline planning, and quote generation.

This project was built to help consultants, developers, freelancers, solution architects, and small businesses estimate work more consistently while keeping everything inside the Microsoft ecosystem.

The Community Edition is shared for learning, experimentation, and adaptation. It was intentionally left without a strict control naming convention inside Power Apps, making it a useful learning resource for people who want to understand how a real application is put together and how different components work together.

## What's Included

This repository contains:

* Power Apps Solution (.zip)
* Sample Service Library Excel file
* Sample Projects Excel file
* Basic setup resources
* Community documentation

The sample Excel files can be used to create or populate the required SharePoint Lists for testing.

## Installation

### Step 1

Import the Power Apps Solution into your Microsoft Power Platform environment.

### Step 2

Create the required SharePoint Lists.

The included sample Excel files can be imported into SharePoint to quickly generate the required list structures.

### Step 3

Import the sample data provided.

This allows you to explore the application immediately without manually entering services or projects.

### Step 4

If required, add any additional SharePoint columns that are documented with the solution.

### Step 5

Open the application and begin testing.

The basic workflow is:

Create Project

↓

Select Services

↓

Generate Estimate

↓

Review Timeline

↓

Review Financials

↓

Generate Quote

## Customisation

The application was designed to be flexible.

You can easily:

* Add new services
* Change hourly rates
* Add new categories
* Modify pricing assumptions
* Adjust project phases
* Expand the service catalogue
* Tailor the solution for your own organisation

The Community Edition is intended as a starting point and can be adapted to suit many different industries.

## About This Release

This is a community release shared to encourage learning and collaboration.

The project is not intended to be a polished commercial product. Instead, it provides a practical example of building business solutions using standard Microsoft Power Platform components without relying on premium connectors.

Many of the internal components retain their original names, allowing learners to follow the development process more naturally and become familiar with how Power Apps solutions are structured.
